## Ecosystem Overview
The user has a massive ecosystem called **SEMSE/Prometeo** that includes:

### 1. SEMSE Project (Construction/Services P2P Platform)
- Marketplace for construction services with escrow payments
- Smart contracting (ADR-001), assisted execution (ADR-002)
- TrustEngine for reputation scoring
- Webhooks system for integrations

### 2. Prometeo Ecosystem (AI/Music/Quantum)
- Motor Prometeo: Embedding engine (64-dim vectors) with Anthropic Claude integration
- Nexus Vector DB: Simulated vector database with k-NN and semantic search
- MCA (Motor de Consciencia): Multi-layer analysis (Perceptual, Cognitive, Emotional, Transcendental)
- MusicGenius: AI music generation platform (12-week MVP plan)

### 3. RAG Architecture (Retrieval-Augmented Generation)
- Embedding generation → Vector storage → Semantic search → LLM augmentation
- 7 key capabilities: Semantic Search, Recommendations, Auto-Clustering, RAG Base, Duplicate Detection, Sentiment Analysis, Multimodal Search

### 4. Architecture Patterns
- Monorepo with Turborepo (83+ modules across 6 phases)
- Microservices: Gateway, Auth, Query, Ingest, Indexer, Graph, Ranker, P2P
- Stack: React/Next.js + Express/FastAPI + PostgreSQL + Redis + Docker/K8s
- Observability: Prometheus, Grafana, Tempo, Loki

### 5. Integration Systems (from sistemas de integracion)
- IntegradorCuantico: Quantum-level data integration
- FusionadorDeConsciencia: Consciousness fusion module
- SintetizadorDeRealidad: Reality synthesis engine
- ConectorUniversal: Universal connection interface

### 6. SEMSE Evolution (v10.1 → v10.3)
- v10.1: UI Base (Fortaleza Soberana)
- v10.2: Motor Prometeo integration (embeddings, cosine similarity, heatmaps)
- v10.3: Nexus Vector DB (k-NN, t-SNE/PCA visualization, RAG simulation)
