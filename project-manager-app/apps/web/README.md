# apps/web

Next.js App Router para portales Cliente/Pro/Ops.

Responsabilidades:
- Autenticación y sesiones de usuario.
- UX de marketplace, FSM, evidencia, pagos y ops.
- Consumo de API REST `/v1` mediante cliente tipado.

No debe contener:
- Lógica de negocio crítica del dominio.
- Acceso directo a base de datos.
