# Skill: Resolución de Disputas

## Proceso de disputa en SEMSE OS

1. **Parte abre disputa** → fondos en escrow quedan congelados
2. **SEMSE notifica** a la otra parte (24h)
3. **Periodo de evidencia**: 72h para que ambas partes suban documentación
4. **Revisión**: SEMSE revisa evidencia (1-3 días hábiles)
5. **Resolución**: SEMSE emite decisión + distribución de fondos
6. **Apelación**: 24h para apelar si hay error factual

## Tipos de disputas comunes

### "Trabajo incompleto"
**Lo que necesita el cliente para ganar:**
- Lista clara de qué falta según el contrato original
- Fotos de qué está sin terminar
- Contrato que especifique ese trabajo

**Lo que necesita el contratista para ganar:**
- Evidencia de que el trabajo sí se completó (fotos con fecha)
- Mensajes del cliente aceptando el trabajo
- Argumento de que lo no hecho no estaba en el alcance

### "Calidad del trabajo"
**Crítico**: la calidad subjetiva es difícil de disputar sin estándar claro
- Si el contrato especificaba estándares (ej. "pintura de primera calidad"), úsalo
- Fotos comparativas (antes/después, vs. estándar acordado)
- En casos extremos: inspector de construcción independiente ($150-$300)

### "Cobro excesivo / trabajo extra no acordado"
- El cliente tiene razón si no hay Change Order firmado
- El contratista puede argumentar "enriquecimiento injusto" si el trabajo fue evidente y aceptado

### "Materiales incorrectos o dañados"
- Fotos del daño o material incorrecto
- Especificación de qué se acordó vs. qué se instaló
- Cotizaciones de reposición

## Evidencia más valiosa en disputas

**Alta credibilidad:**
- Fotos/videos con metadata de fecha/hora
- Mensajes de WhatsApp/text con timestamps
- Contrato firmado digitalm
- Facturas de materiales
- Inspecciones pasadas/falladas

**Baja credibilidad:**
- Testimonios verbales sin respaldo
- Fotos sin fecha o de baja calidad
- "El dijo/ella dijo" sin documentación

## Escalamiento fuera de SEMSE

### Small Claims Court (Corte de Pequeñas Reclamaciones)
- **Límite**: $5,000-$25,000 según estado
- **Costo**: $30-$100 para presentar
- **Tiempo**: 1-3 meses
- **Sin abogado necesario** (recomendado)

### Mediation privada
- **Costo**: $150-$500/hora (dividido entre partes)
- **Tiempo**: 1-4 horas
- **Resultado**: acuerdo vinculante si ambas partes aceptan

### Contractor's License Board
- Si el contratista tiene licencia estatal, se puede presentar queja formal
- Puede resultar en multa o suspensión de licencia
- Útil como presión para llegar a acuerdo

## Resoluciones típicas

| Tipo | Resultado más común |
|------|-------------------|
| Trabajo incompleto con evidencia | 70-80% del contrato liberado al contratista, resto retenido |
| Calidad deficiente sin estándar claro | 50-50 o 60-40 a favor del contratista |
| No aparición (abandonment) | 90-100% devuelto al cliente |
| Extra work sin change order | Negociación: 40-60% del costo extra al contratista |
