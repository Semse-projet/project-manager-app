# 05 — Module Flows

## Flujo estándar al seleccionar módulo

```txt
Mission Control → Module Hub → Submodule Dashboard → Entity Detail → Tabs / Actions
```

## WorkOps

```txt
Mission Control → WorkOps → Field Ops → Job Detail → Tasks / Evidence / Finance / Documents / Activity
```

## Finance

```txt
Mission Control → Finance → Dashboard → Escrow → Escrow Account → Transactions / Release / Approval
```

## Trust

```txt
Mission Control → Trust → Dashboard → Reputation → Company Profile → Credentials / Reviews / Risk
```

## Intelligence

```txt
Mission Control → Intelligence → Prometeo / Agents / RAG → Agent Detail → Runs / Tools / Memory / Logs
```

## Tool Hub

```txt
Mission Control → Tool Hub → App Tile → Context Bridge → Copy Prompt / Open External Tool / Terminal Sync
```

## Verticals

```txt
Mission Control → Verticals → Construction / Turnovers / Cleaning / Agro / Maintenance → Project / Unit Detail
```
