# UI Click Flow QA

## Flujo 1 — Mission Control a WorkOps

1. Abrir `/admin/mission-control`.
2. Ver tarjeta WorkOps.
3. Click en Open Module.
4. Debe abrir `/admin/workops`.
5. Ver cards: Field Ops, Workers, Contractors, Change Orders, PMO, QA.
6. Click Field Ops.
7. Debe abrir `/admin/field-ops`.

## Flujo 2 — Mission Control a Tool Hub

1. Abrir `/admin/mission-control`.
2. Click Tool Hub.
3. Debe abrir `/admin/tool-hub`.
4. Ver tiles de herramientas externas.
5. Ver Context Bridge.
6. Click Copy Prompt.
7. Debe copiar texto o mostrar fallback.

## Flujo 3 — Mission Control a Verticals

1. Abrir `/admin/mission-control`.
2. Click Verticals.
3. Debe abrir `/admin/verticals`.
4. Ver Construction, Property Turnovers, Cleaning, Agro/FarmOps, Maintenance.

## Flujo 4 — Mobile

1. Abrir cada hub en ancho 390px.
2. Validar que no haya scroll horizontal innecesario.
3. Validar botones visibles.
