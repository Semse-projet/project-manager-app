## Reglas

- No usar logos oficiales en la primera fase.
- No guardar tokens.
- No crear OAuth todavía.
- Los links pueden salir a URLs configurables o placeholders seguros.
- La acción Copy Context debe copiar el prompt del Context Bridge.
