# UI QA Checklist

## Desktop

- [ ] Sidebar visible.
- [ ] Estado activo correcto.
- [ ] Cards alineadas.
- [ ] Textos legibles.
- [ ] No hay overflow horizontal.
- [ ] CTAs claros.

## Mobile

- [ ] Header no se rompe.
- [ ] Cards se apilan correctamente.
- [ ] Botones son tocables.
- [ ] Texto no se corta de forma crítica.

## Dark theme

- [ ] Contraste suficiente.
- [ ] Bordes y fondos consistentes.
- [ ] Estados de alerta distinguibles.

## Navegación

- [ ] Mission Control → módulo.
- [ ] Módulo → submódulo.
- [ ] Submódulo legacy funciona.
- [ ] Back/forward del navegador funciona.
