# Repository Rules

Esta carpeta concentra las reglas que ordenan el repositorio `labsemse`.

## Índice

- `CANONICITY.md`
- `ARCHIVE_POLICY.md`
- `CONTRIBUTING.md`
- `MIGRATION_RULES.md`
- `OPERACION_ASISTIDA_RETENTION_POLICY.md`

## Alcance explícito

Estas reglas también gobiernan la documentación y trazabilidad de la capa de operación asistida del ecosistema.

Eso incluye:

- patrones absorbidos desde el entorno del operador;
- referencias aterrizadas en `agents/references/`;
- reportes fechados en `reportes/`;
- su reflejo en `constitution/` y `program/`.

## Regla

Aquí viven normas de precedencia, archivo, contribución y migración.

No es visión de producto ni constitución del dominio. Es gobierno del repositorio.
