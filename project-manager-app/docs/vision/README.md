# Vision

Esta carpeta guarda la direccion estrategica del sistema.

No define tareas de implementacion detalladas.
Define:

- que estamos construyendo;
- por que existe;
- que decisiones ya no deben reabrirse facilmente;
- cual es la relacion entre SEMSE, Trust, Ops y Prometeo;
- que debe preservarse aunque cambie el detalle tecnico.

## Regla

La vision cambia lento.

Si una idea nueva contradice estos documentos, primero debe explicarse por que mejora la direccion general.

No deben vivir aqui:

- reportes de ejecucion;
- backlog o sprints;
- decisiones tecnicas detalladas por archivo;
- evidencia de validacion.
