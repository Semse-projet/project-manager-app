# Spec: M3.2 Extended Metrics for ProTools Trades

## Implementation Status

This future-phase spec is approved as the implementation contract. The corresponding Master Plan blocks remain `PENDING` until every targeted trade engine exposes the required metrics and tests.

## Problem Statement

All ProTools trade engines need consistent confidence, dispute risk, production schedule, hidden damage and algorithm trace outputs so estimates can support evidence, change orders and payment readiness.

## Scope

- In scope:
  - Confidence score for all trade engines.
  - Dispute risk score for all trade engines.
  - Production schedule for structural trades.
  - Hidden damage assessment for plumbing, HVAC, siding and electrical.
  - Standard algorithm trace across all engines.
- Out of scope:
  - Paid labor calibration data; see M3.3.

## Tests Required

- [x] Existing extended metrics unit coverage.
- [x] Every active trade returns `algorithmTrace`.
- [x] Every active trade returns confidence/dispute risk where applicable.
- [x] Structural trades return production schedule.
