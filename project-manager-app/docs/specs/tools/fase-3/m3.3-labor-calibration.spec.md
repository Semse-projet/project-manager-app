# Spec: M3.3 Labor Calibration

## Implementation Status

This future-phase spec is approved as the implementation contract. The corresponding Master Plan blocks remain `PENDING` until licensing decisions, calibration fixtures and tests are delivered.

## Problem Statement

Labor estimates need calibration against recognized trade units and real completed project outcomes. Without calibration, SEMSE risks confident but inaccurate estimates.

## Scope

- In scope:
  - Evaluate RSMeans Complete vs NECA/PHCC/SMACNA sources.
  - Calibrate electrical labor units.
  - Calibrate plumbing and HVAC units.
  - Material waste factors.
  - Validation against completed project outcomes.
- Out of scope:
  - Scraping licensed data.
  - Publishing proprietary rate tables without license review.

## Required Gates Before Implementation

- Decide licensed data source.
- Document license constraints.
- Define calibration fixture format.
- Define how project outcome data is anonymized.

- [x] Existing labor engine unit tests.
- [x] Calibration fixtures for electrical.
- [x] Calibration fixtures for plumbing/HVAC.
- [x] Waste factor tests by material class.
