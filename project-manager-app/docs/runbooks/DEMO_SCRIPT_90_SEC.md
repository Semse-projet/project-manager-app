# SEMSE Demo Script — 90 Seconds

## Apertura

SEMSE es una capa operativa para trabajos reales.
No solo publica jobs: conecta ejecución, evidencia, escrow y manejo de excepciones en un solo flujo.

## 1. Overview

En esta pantalla se ve la lectura mínima del negocio.
La idea central es simple: SEMSE organiza el trabajo alrededor de jobs reales y su lifecycle operativo.

## 2. Happy path

Entramos a un job activo.
Aquí vemos milestones que pueden pasar por submit, approve y release.
Eso significa que la ejecución y el control financiero están conectados.

## 3. Evidence

Cada milestone puede dejar evidencia verificable.
Eso reduce fricción, mejora auditoría y acelera aprobación.

## 4. Escrow

El dinero no se mueve suelto.
Se administra dentro del contexto del job y del avance real del trabajo.

## 5. Dispute

Y si algo sale mal, SEMSE también muestra y opera excepciones.
No solo sirve para el happy path; también sirve para gobernar conflicto y resolución.

## Cierre

Esta demo no intenta mostrar todo el universo SEMSE.
Muestra que el flujo principal ya puede presentarse de forma clara, reproducible y operable.
