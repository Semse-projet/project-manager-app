## Nota de estado transversal

Cuando una capacidad transversal como la operación asistida cambie de forma material:

- el cambio estructural se refleja en `constitution/` o `program/`;
- la evidencia y la intervención concreta se registra en `reportes/`;
- si afecta implementación, debe aterrizarse también en backlog o mapping técnico del monorepo.
