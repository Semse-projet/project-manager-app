# Governance

Esta carpeta concentra material de gobierno operativo y de estructura del repositorio.

## Subáreas

- `SEMSE_PERMISSION_MATRIX.md`
  Matriz de permisos del ecosistema.
- `OPERACION_ASISTIDA_TRACEABILITY_MAP.md`
  Mapa formal de trazabilidad de la capa de operación asistida.
- `repository-consolidation/`
  Auditorías y análisis históricos sobre consolidación, canonicidad y absorción de valor entre carpetas.

## Regla

- decisiones vigentes de soberanía y precedencia pueden vivir en raíz;
- la trazabilidad formal de capacidades transversales puede vivir aquí cuando cruza canon, programa, reglas y evidencia;
- análisis de consolidación del repositorio deben vivir aquí, no en la raíz.
