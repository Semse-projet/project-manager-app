# Distillation

Esta carpeta concentra la disciplina de destilación desde satellites o fuentes externas hacia el canónico.

## Índice

- `DISTILLATION_QUEUE.md`
- `DISTILLATION_LOG.md`
- `DISTILLATION_CLAUDE_CODE_INSIGHTS.md`

## Regla

Aquí viven:

- cola de valor pendiente;
- historial de lo ya absorbido;
- insights aplicables al canónico.

No deben vivir aquí logs de sesión ni estado global del ecosistema.
