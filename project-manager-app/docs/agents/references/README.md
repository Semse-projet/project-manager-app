# Referencias de Agentes

Esta carpeta agrupa análisis de sistemas externos que sirven como fuente de patrones para `SEMSE`, pero que no constituyen documentación canónica del producto por sí mismos.

## Regla

- si el documento analiza una fuente externa y aterriza decisiones para `SEMSE`, vive aquí;
- si el documento ya define la arquitectura oficial de `SEMSE`, vive en otra carpeta de `agents`;
- si el documento solo registra una ejecución o un cierre, vive en `reportes`.
