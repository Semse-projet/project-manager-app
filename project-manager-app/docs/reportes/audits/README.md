# Audits

Esta carpeta agrupa auditorías legacy o externas que no pertenecen al ciclo principal de cierres técnicos recientes.

## Qué sí vive aquí

- auditorías heredadas;
- reportes externos;
- diagnósticos puntuales que no forman parte del hilo principal de ejecución reciente.

## Qué no vive aquí

- arquitectura canónica;
- cierres técnicos corrientes del trabajo activo;
- prompts;
- planning.

## Regla

Si un documento existe para decir "qué se auditó y qué riesgos se detectaron", puede vivir aquí.
Si además se convierte en criterio estable del sistema, su versión canónica debe promocionarse a `repository-rules/`, `program/`, `agents/`, `vision/` o `constitution/`.
