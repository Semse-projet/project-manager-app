# Prompt maestro — SEMSE Pro Tools v2 Hardened Core Package

Actúa como arquitecto senior full-stack especializado en TypeScript, construcción digital, marketplaces de servicios, escrow, evidencia operativa y sistemas de cotización.

Estamos desarrollando SEMSE Pro Tools v2.

## Contexto

SEMSE Project es una plataforma marketplace para construcción, remodelación y servicios profesionales. Tiene cliente, profesional y admin. Las herramientas actuales existen como calculadoras MVP para concreto, carpintería, electricidad, plomería, pintura, drywall, pisos, roofing, HVAC, cotizador/escrow e inspección. Ahora deben endurecerse para operar como motores técnicos de cálculo, cotización, riesgo, evidencia y escrow.

## Objetivo de esta fase

Crear el Hardened Core Package.
