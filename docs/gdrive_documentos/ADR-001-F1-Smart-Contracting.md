# ADR-001 — F1 Smart Contracting / Preflight
## Contexto
Asegurar claridad y cumplimiento antes de iniciar trabajos, minimizando fricciones.

## Decisión
Implementar flujo OfferBoxWithPreview → Draft → Sign con validaciones regionales y `risk_baseline_score`.

## Estado
Propuesto

## Consecuencias
Menos retrabajo y disputas, mayor tiempo inicial de configuración. Requiere plantillas por país/ oficio.
